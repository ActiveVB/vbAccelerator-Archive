Collapsing and Expanding Bars is slow with a large number of items and a big control.

Summary

Id:12138.16
Type: Issue
Current Status:Resolved

Detail

10 Nov 2003OpenSteve McMahon

The speed of the collapse and expand animation is rate limited to 16 pixels. Also, the initial scrolling amount is set to 1 pixel. This needs to be adjusted for larger bars so that there is no limitation and the initial scroll speed is increased.

11 Nov 2003ResolvedSteve McMahon

Removed rate limiter for scroll speed and set some more reasonable initial scroll sizes for larger bars.
