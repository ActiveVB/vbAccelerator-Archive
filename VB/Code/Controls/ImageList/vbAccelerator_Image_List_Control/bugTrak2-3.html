Images Property Page renders incorrectly in VB6 when a Manifest is applied

Summary

Id:2.3
Type: Bug
Current Status:Open

Detail

11 Nov 2003OpenSteve McMahon

In VB6, controls shown directly in Frames don’t draw properly when a manifest is applied. This doesn’t affect the working of the control but the Bitmap Transparent Colour options cannot be read under VB6.
