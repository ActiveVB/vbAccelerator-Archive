Error on Pre-XP Machine

Summary

Id:1414.1
Type: Bug
Current Status:Resolution Identified

Detail

12 Jan 2004Resolution IdentifiedArnout de Vries

An error occurs when using the vbAccelerator Progress Bar Class on a non XP machine and setting the xpstyle to true.

All that is needed is some small error trapping when querying for the current theme, because of the fallback routine later on when bDrawn is false.

Function DrawToDC:

  ' Draw background:
   If Not m_picBack Is Nothing Then
      If m_eBackPictureMode = EVPRGTile Then
         TileArea _
            lHDC, _
            0, 0, _
            lWidth, lHeight, _
            m_picBack.hDC(m_picBack.Width, m_picBack.Height), _
            m_picBack.Width, m_picBack.Height, _
            0, 0
      Else
         StretchBlt _
            lHDC, _
            0, 0, lWidth, lHeight, _
            m_picBack.hDC(m_picBack.Width, m_picBack.Height), _
            0, 0, _
            m_picBack.Width, m_picBack.Height, _
            vbSrcCopy
      End If
   Else
      If (m_bXpStyle) Then
        On Error Resume Next ' Will prevent non XP from crashing
         hTheme = OpenThemeData(hwnd, StrPtr("Progress"))
