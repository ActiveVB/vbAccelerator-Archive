Status Bar SizeGrip Property does not work correctly at runtime

Summary

Id:1417.2
Type: Bug
Current Status:Open

Detail

1 Feb 2003OpenSergey Pitutin

Current code reads:

If (bSizeGrip) Then lStyle = lStyle And SBARS_SIZEGRIP Else lStyle = lStyle And Not SBARS_SIZEGRIP End If

It should read:

If (bSizeGrip) Then lStyle = lStyle Or SBARS_SIZEGRIP Else lStyle = lStyle Xor SBARS_SIZEGRIP End If
