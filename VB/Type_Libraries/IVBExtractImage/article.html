VB IExtractImage Interface (IVBExtractImageLib.tlb)

This type library provides access to the IExtractImage object exposed by the IShellFolder interface, which is used to create thumbnails from objects.

Supported calls:

  • IMalloc (as IVBMalloc)
  • IEnumIDList (as IVBEnumIDList)
  • IShellFolder (as IVBShellFolder)
  • IExtractImage

The library also includes the IIDFromString OLE32.DLL function.

To see the type library in use, check out the Thumbnail Extractor sample, which uses it to obtain a thumbnail images for a selected file.
